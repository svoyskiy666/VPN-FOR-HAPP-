## Discord и YouTube не открываются?

1. **Обнови подписку** в Happ (потянуть вниз / кнопка обновления)
2. Выбери сервер **🇫🇮 Финляндия**, **🇳🇱 Нидерланды** или **🇩🇪 Германия** (не США и не «другие»)
3. После обновления должен подтянуться роутинг **svoyskiy.ru** (весь трафик + DNS Cloudflare через VPN)
4. Переподключи VPN (выкл → вкл)
5. Если не помогло — попробуй следующую ноду FI/NL/DE

В подписке уже включено:
- глобальный прокси + Cloudflare DNS (чтобы Discord/YouTube не резолвились через РКН)
- правила для `youtube` / `discord` / `google`
- фрагментация TLS (обход DPI)
- брендинг **svoyskiy.ru** в приложении

## Как подключить

1. Сделай репозиторий **Public**
2. Установи Happ — [happ.su](https://www.happ.su/)
3. **+** → добавить подписку → ссылка выше
4. Обнови → выбери **FI / NL / DE** → подключись
5. В Happ будет написано **VPN от svoyskiy.ru**, кнопка на сайт [svoyskiy.ru](https://svoyskiy.ru)
